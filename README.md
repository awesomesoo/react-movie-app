# 💜 Movie App

React JS Fundamentals Course movie app

### 컴포넌트 종류

#### Navigation

화면이동을 위한 컴포넌트.

#### Movie

영화에 대한 정보를 담은 컴포넌트.
id, year, title, summary, poster, genres

#### Home

홈페이지의 첫화면을 구성한는 컴포넌트.

#### Detail

영화 리스트를 보여주는 컴포넌트.

#### About

서브페이지를 생성하여 화면 전환을 보여주는 컴포넌트.
